# Screen Mirror - 投屏工具

Android USB/无线投屏工具，基于 scrcpy。

## 功能

- USB 投屏
- 无线调试投屏
- 亮屏/锁屏
- Home/返回键
- 音量调节

## 使用方法

1. 下载 Mirror.exe 和 scrcpy 文件夹
2. 将它们放在同一目录下
3. 双击 Mirror.exe 运行
4. USB 投屏：连接手机后点击"USB投屏"
5. 无线投屏：输入 IP:PORT 后点击"无线投屏"
